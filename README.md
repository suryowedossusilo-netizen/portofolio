# 🌐 Personal Portfolio Website

## 📌 Features
- Responsive portfolio design
- Project showcase section
- Skills and experience section
- Contact page
- Smooth animations and modern UI

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript
